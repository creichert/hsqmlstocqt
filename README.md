# Build
```
    cabal sandbox init
    cabal sandbox install --dependencies-only
    cabal build
```

This is a Haskell version of the Qt Quick example
[StocQt](http://qt-project.org/doc/qt-5/qtquick-demos-stocqt-example.html).
